# Aelunor Animation Pipeline (Motion Canvas)

Prototype for building UI animation assets from existing Aelunor sprite
sheets — TypeScript scenes, headless rendering, no diffusion models, no
ComfyUI, no browser clicking.

```powershell
cd 01_repo/aelunor-core/tools/animation-pipeline
npm install
npm run build:assets
```

Full documentation: [`docs/ASSET_ANIMATION_PIPELINE.md`](../../docs/ASSET_ANIMATION_PIPELINE.md)
