# Narrator Prompt

Aktive Prompt-Logik liegt im Core-Backend, nicht in diesem Platzhalter-Dokument.

## Pflegeziel

Prompt-Dokumente sollen kuenftig die produktive Prompt-Quelle referenzieren oder erklaeren, welche Prompt-Datei im Code genutzt wird. Tests duerfen echte Narrator-Aufrufe nicht ausloesen.

## Regel

Prompt-Aenderungen brauchen mindestens einen Fake-Narrator-Testplan fuer Turn-/Canon-/Patch-Verhalten.
