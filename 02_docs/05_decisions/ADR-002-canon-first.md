# ADR-002 Canon First

## Status

Accepted.

## Entscheidung

Persistenter Campaign State ist die Wahrheit. Freier Story-Text ist wichtig, ersetzt aber keine strukturierten State-Aenderungen.

## Konsequenzen

- Relevante Welt-, Charakter-, Progression- und Codex-Aenderungen muessen strukturiert landen.
- Canon-Gate darf unsichere Aenderungen skippen oder flaggen.
- UI darf keine parallele Wahrheit erfinden.
