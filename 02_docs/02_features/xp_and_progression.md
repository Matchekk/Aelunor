# XP and Progression

XP und Progression sind MVP-relevant, aber duerfen den Story-Kern nicht ueberladen.

## Aktueller Stand

- Character Sheets enthalten Class-/Skill-/Progression-Snapshots.
- Turn-Patches koennen Progression sichtbar machen.
- Canon-Gate entscheidet ueber sichere strukturierte Aenderungen.

## Regeln

- Progression braucht sichtbare Story-Begruendung oder strukturierte Quelle.
- Balancingwerte moeglichst datengetrieben halten.
- UI soll Progression erklaeren, aber nicht eine zweite Wahrheit fuehren.
