# Races

Races sind Teil der Welt- und Codex-Struktur.

## Aktueller Stand

- Backend-State enthaelt `world.races`, `race_alias_index` und `codex.races`.
- Drawers koennen Codex-Eintraege fuer Races oeffnen.
- Setup/World Generation kann Races vorbereiten.

## Regel

Race-Wissen ist spielrelevant, aber der sichtbare Wissensstand muss ueber Codex/Knowledge-Level kontrolliert bleiben.
