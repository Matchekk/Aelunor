# Classes

Klassen sind Teil des Character-/Progression-Systems.

## Aktueller Stand

- Class-Snapshots sind im Frontend-Vertrag `CharacterClassSnapshot` modelliert.
- Character Drawers zeigen aktuelle Klasse, Rang, Level und Ascension-nahe Felder.
- Progression-Patches koennen Klassenentwicklung betreffen.

## Regel

Klassen duerfen nicht nur UI-Labels sein. Relevante Klassenentwicklung muss im Campaign State oder in Character Sheets persistiert sein.
