# Skills

Skills sind Charakterfaehigkeiten und Teil der strukturierten Progression.

## Aktueller Stand

- Frontend-Vertrag: `CharacterSkillSnapshot`
- Skill-Progression kann ueber Turn-/Canon-Patches entstehen.
- Character Drawer ist die primaere Detailansicht.

## Regeln

- Skill-Freischaltung braucht strukturierte Daten.
- Unsichere Extractor-Ergebnisse duerfen nicht still committed werden.
- Skill-/Class-Affinitaeten sollten datengetrieben bleiben.
