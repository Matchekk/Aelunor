# Naming Policy

## Produktname

Der Produktname ist `Aelunor`.

## Sichtbare UI

- Neue sichtbare Texte sollen Aelunor, Campaign, Session, Story, Canon, Codex, Party, Scene und Character konsistent verwenden.
- Alte Isekai-Prototyp-Namen nicht in neue UI-Texte uebernehmen.

## Code

Historische Storage Keys oder Legacy-Kompatibilitaetsnamen duerfen bleiben, wenn eine Umbenennung Migrationen erfordern wuerde. Neue Module sollen fachliche Namen verwenden.
