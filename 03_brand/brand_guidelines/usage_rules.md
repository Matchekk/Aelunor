# Usage Rules

- Brand-Quellen liegen unter `03_brand/`.
- Produktiv ausgelieferte Assets liegen im Core unter `01_repo/aelunor-core/app/static/brand/`.
- Referenz-Wallpaper koennen fuer UI-Stimmung genutzt werden, aber UI-Text und Controls bleiben code-native.
- Assets nicht direkt aus `03_brand/` zur Laufzeit referenzieren, wenn die App aus Docker/Core ausgeliefert wird.
